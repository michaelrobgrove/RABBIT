## üõ†Ô∏è Installation & Usage

### 1. Hosted Version (Easiest)
Simply visit our hosted version: **[Launch RABBIT](https://rabbit-30m.pages.dev)**

*Note: If your IPTV service uses HTTP (not HTTPS) and doesn't work, try the self-hosted option.*

### 2. Self-Hosted (Recommended for HTTP Services)
Running RABBIT locally is the best way to ensure compatibility with all IPTV services.

1.  Download the repository.
2.  Run `python -m http.server 8000` (or any web server).
3.  Open `http://localhost:8000`.

**Full Guide:** [SELF_HOSTING.md](SELF_HOSTING.md)
